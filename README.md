# For-Cursor

Dépôt utilisé uniquement comme espace ASC (assistant/chat), sans code applicatif.

Règles appliquées (voir `.cursor/rules/asc.mdc`) :

- Réponses **toujours en français**.
- Seul le message de l'utilisateur est utilisé comme contexte.
- Aucun parcours automatique du dépôt ni ajout de contexte issu du codebase.
